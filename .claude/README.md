# Claude Configuration for ExpenseTracker

This directory contains Claude-specific configuration and context files for the ExpenseTracker project.

## Files

- `claude.md` - Main project context and instructions for Claude
- `commands/` - Custom slash commands for Claude Code

## Usage

Claude Code automatically reads files in this directory to understand the project context and provide better assistance.

## Commands

- `/document-feature <feature-name>` - Generate both technical and user documentation for a feature
