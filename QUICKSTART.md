# Quick Start Guide - ExpenseTracker

Get up and running with ExpenseTracker in 3 simple steps!

## Step 1: Start the Application

The development server is already running! Open your browser and visit:

```
http://localhost:3000
```

If you need to start it manually:
```bash
cd expense-tracker
npm run dev
```

## Step 2: Add Your First Expense

1. Click **Expenses** in the top navigation
2. Click the **Add Expense** button
3. Fill in the form:
   - Date: Today's date (default)
   - Amount: e.g., 15.99
   - Category: Food
   - Description: "Lunch at cafe"
4. Click **Add Expense**

## Step 3: View Your Dashboard

1. Click **Dashboard** in the navigation
2. See your expense summary and analytics

## What You Can Do

### Add Expenses
- Track all your spending with detailed information
- Categorize into 6 different categories
- Add descriptions for easy tracking

### Analyze Spending
- View total and monthly expenses
- See which category you spend the most on
- Track trends over 6 months
- Review category breakdowns

### Manage Expenses
- Search expenses by description
- Filter by category and date range
- Edit existing expenses
- Delete expenses (with confirmation)

### Export Data
- Download your expenses as CSV
- Works with current filters
- Open in Excel or Google Sheets

## Pro Tips

1. Add expenses as they happen for accuracy
2. Use clear, searchable descriptions
3. Be consistent with categories
4. Review your dashboard weekly
5. Export data monthly for backup
